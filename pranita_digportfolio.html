AI Recipe Generator (Mise en Place)

An AI-powered recipe generation platform that creates personalized recipes using available ingredients, cuisine preferences, dietary restrictions, and cooking constraints. The application integrates Groq AI to generate complete recipes with ingredients, cooking instructions, chef tips, and nutrition information.


📌 Phase 1 – Project Planning & Requirement Analysis

Project Overview

The project aims to help users prepare meals using ingredients already available at home through AI-powered recipe generation.

Problem Statement

Many people have ingredients but struggle to decide what meals can be prepared. Existing recipe platforms require manual searching and do not provide personalized recommendations.

Objectives

  • Generate recipes using Artificial Intelligence.
  • Reduce food wastage.
  • Provide personalized cooking suggestions.
  • Support cuisine and dietary preferences.
  • Create an easy-to-use cooking assistant.

Requirements

  • Ingredient input system
  • Cuisine selection
  • Diet selection
  • Recipe generation module
  • Nutrition display
  • Copy and print functionality

📌 Phase 2 – System Design & Architecture

Technology Stack

  • HTML5
  • Tailwind CSS
  • JavaScript (ES6)
  • Groq AI API

System Architecture

User Input → Prompt Builder → Groq AI API → JSON Parser → Recipe Renderer → Recipe Output

Modules

  • User Input Module
  • Prompt Builder Module
  • AI Integration Module
  • JSON Parsing Module
  • Recipe Display Module

Workflow

Users enter ingredients and preferences. The application generates a structured prompt, sends it to the Groq AI model, processes the response, and displays a complete recipe card.


📌 Phase 3 – Development & Implementation

Features Developed

  • Ingredient-based recipe generation
  • Cuisine and diet filters
  • Serving size customization
  • Cooking time constraints
  • Recipe regeneration
  • Copy recipe feature
  • Print recipe functionality

Key Functions

  • generate() – Generates recipes.
  • buildPrompt() – Creates AI prompts.
  • callGroq() – Connects with Groq API.
  • renderRecipe() – Displays recipes.
  • copyRecipe() – Copies recipe text.

Testing

  • Functional Testing
  • API Testing
  • UI Testing
  • Responsive Design Testing
  • Error Handling Testing
